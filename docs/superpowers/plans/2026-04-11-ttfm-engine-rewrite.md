## Key Design Decisions

1. **TF Pairs** — TTFM auto-mapping: `15M_1M`, `30M_3M`, `1H_5M`, `4H_15M`
2. **No RTH filter** — all session hours included
3. **No risk profiles** — single output per model (no structural/split/simple_1r)
4. **No WIN/LOSS** — trades have MAE/MFE only, no outcome or R-multiple
5. **No SMT** — TTFM doesn't have it
6. **Body for confirmation** — pivots use `max(open,close)` / `min(open,close)`
7. **MFE cutoff** — end of next HTF candle close after entry
8. **One trade per T-Spot** — after confirmation, reset touch state
9. **CISD** — series extreme (already implemented), drawn as reference, not used for entry

Port directly from TTFM Pine `calculateLogMidpoint`:

```python
import math

def log_midpoint(h, l, o, c):
    """
    TTFM log-space midpoint. If the dominant wick > body, midpoint anchors
    to the midpoint of that wick. Otherwise simple log-midpoint of range.
    """
    log_h = math.log(h)
    log_l = math.log(l)
    log_o = math.log(o)
    log_c = math.log(c)

    body = abs(log_c - log_o)
    upper_wick = log_h - max(log_o, log_c)
    lower_wick = min(log_o, log_c) - log_l

    if max(upper_wick, lower_wick) > body:
        if upper_wick > lower_wick:
            log_mid = log_h - upper_wick / 2
        else:
            log_mid = log_l + lower_wick / 2
    else:
        log_mid = (log_h + log_l) / 2

    return math.exp(log_mid)
```

```python
def detect_tspot(lc, pc, ppc):
    """
    Detect T-Spot from 3 consecutive closed HTF candles.
    lc/pc/ppc are dicts with keys: o, h, l, c
    Returns (tspot_type, direction, sweep_mid, close_level, c2_level, c2_candle_idx)
    or None if no T-Spot.

    Types checked in order (first match wins):
      1. Normal Bearish    2. Normal Bullish
      3. Expansive Bearish 4. Expansive Bullish
      5. Pro-trend Bullish 6. Pro-trend Bearish
    """
    outside = is_outside_bar(lc['h'], lc['l'], lc['c'], pc['h'], pc['l'])
    if outside:
        return None

    mid_prev = log_midpoint(pc['h'], pc['l'], pc['o'], pc['c'])

    # Type 1: Normal Bearish
    if lc['h'] > pc['h'] and lc['c'] < pc['h']:
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] < sweep_mid:
            return ('normal', 'SHORT', sweep_mid, lc['c'], lc['h'], 'lc')

    # Type 2: Normal Bullish
    if lc['l'] < pc['l'] and lc['c'] > pc['l']:
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] > sweep_mid:
            return ('normal', 'LONG', sweep_mid, lc['c'], lc['l'], 'lc')

    # Type 3: Expansive Bearish
    if (pc['h'] > ppc['h']
        and lc['c'] < max(pc['o'], pc['c'])
        and (pc['c'] >= log_midpoint(pc['h'], pc['l'], pc['o'], pc['c'])
             or pc['c'] >= ppc['h']
             or (pc['h'] > ppc['h'] and pc['l'] < ppc['l']
                 and pc['c'] > ppc['l'] and pc['c'] < ppc['h']))):
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] < sweep_mid:
            return ('expansive', 'SHORT', sweep_mid, lc['c'], pc['h'], 'pc')

    # Type 4: Expansive Bullish
    if (pc['l'] < ppc['l']
        and lc['c'] > min(pc['o'], pc['c'])
        and (pc['c'] <= log_midpoint(pc['h'], pc['l'], pc['o'], pc['c'])
             or pc['c'] <= ppc['l']
             or (pc['h'] > ppc['h'] and pc['l'] < ppc['l']
                 and pc['c'] > ppc['l'] and pc['c'] < ppc['h']))):
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] > sweep_mid:
            return ('expansive', 'LONG', sweep_mid, lc['c'], pc['l'], 'pc')

    # Type 5: Pro-trend Bullish
    if (lc['l'] < mid_prev and lc['l'] > pc['o'] and lc['c'] > pc['h']):
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] > sweep_mid:
            return ('protrend', 'LONG', sweep_mid, lc['c'], None, None)

    # Type 6: Pro-trend Bearish
    if (lc['h'] > mid_prev and lc['h'] < pc['o'] and lc['c'] < pc['l']):
        sweep_mid = log_midpoint(lc['h'], lc['l'], lc['o'], lc['c'])
        if lc['c'] < sweep_mid:
            return ('protrend', 'SHORT', sweep_mid, lc['c'], None, None)

    return None
```

```python
def detect_pivot_high(opens, closes, highs, idx):
    """
    2-bar left, 2-bar right pivot high using body (max of open, close).
    Returns the pivot level at idx if idx is a pivot high, else None.
    The pivot is confirmed 2 bars later (caller checks at idx+2).
    """
    if idx < 2 or idx + 2 >= len(opens):
        return None
    body_mid = max(opens[idx], closes[idx])
    # Check 2 bars left and 2 bars right — all must have lower body tops
    for offset in [-2, -1, 1, 2]:
        neighbor_body = max(opens[idx + offset], closes[idx + offset])
        if neighbor_body >= body_mid:
            return None
    return float(body_mid)


def detect_pivot_low(opens, closes, lows, idx):
    """
    2-bar left, 2-bar right pivot low using body (min of open, close).
    Returns the pivot level at idx if idx is a pivot low, else None.
    """
    if idx < 2 or idx + 2 >= len(opens):
        return None
    body_mid = min(opens[idx], closes[idx])
    for offset in [-2, -1, 1, 2]:
        neighbor_body = min(opens[idx + offset], closes[idx + offset])
        if neighbor_body <= body_mid:
            return None
    return float(body_mid)
```

```python
def check_touch(bar_high, bar_low, bar_open, bar_close, touch_level, direction):
    """
    Check if a chart-TF bar touches the T-Spot close level.
    Bullish: price dips to touch_level but closes above it.
    Bearish: price rises to touch_level but closes below it.
    """
    if direction == 'LONG':
        # Bullish T-Spot: touch_level = close (upper edge of box)
        return (bar_low < touch_level or bar_open < touch_level) and bar_close > touch_level
    else:
        # Bearish T-Spot: touch_level = close (lower edge of box)
        return (bar_high > touch_level or bar_open > touch_level) and bar_close < touch_level
```

- [ ] **Step 2: Add confirmation detection function**

```python
def check_confirmation(bar_open, bar_close, pivot_level, pivot_bar_idx,
                       touch_bar_idx, touch_level, direction):
    """
    Check if a chart-TF bar confirms the T-Spot sweep.
    Bullish: open < pivot_high AND close > pivot_high, pivot above touch_level.
    Bearish: open > pivot_low AND close < pivot_low, pivot below touch_level.
    """
    if pivot_bar_idx >= touch_bar_idx:
        return False  # pivot must form BEFORE touch

    if direction == 'LONG':
        return (bar_open < pivot_level
                and bar_close > pivot_level
                and pivot_level > touch_level)
    else:
        return (bar_open > pivot_level
                and bar_close < pivot_level
                and pivot_level < touch_level)
```

### Task 5: Rewrite Main Scan Loop

**Files:**
- Modify: `Fractal Sweep/model_stats.py`

- [ ] **Step 1: Write `scan_ttfm_model()` — the core scan function**

This replaces `detect_setups_base()`. The algorithm:

```
For each HTF bar i (from i=3 onwards):
    When HTF bar i starts (bar i-1 just closed):
        lc = HTF[i-1], pc = HTF[i-2], ppc = HTF[i-3]
        tspot = detect_tspot(lc, pc, ppc)
        if tspot found:
            Reset touch state
            Find chart-TF bars within this HTF period
            Run pivot tracking + touch + confirmation on those bars
            If confirmed:
                Record trade with entry, stop, MAE, MFE
```

```python
def scan_ttfm_model(htf_arrs, chart_arrs, m1_arrs, model_key, cfg):
    """
    Full TTFM scan: T-Spot → Touch → Pivot Sweep → MAE/MFE.

    htf_arrs: numpy arrays for HTF bars (sweep timeframe)
    chart_arrs: numpy arrays for chart-TF bars (confirmation timeframe)
    m1_arrs: numpy arrays for 1m bars (MAE/MFE tracking)
    """
    htf_n = len(htf_arrs['ts_ns'])
    htf_period_ns = cfg['htf_min'] * 60 * 10**9
    trades = []

    for i in range(3, htf_n):
        # Skip gaps (weekends, holidays)
        if htf_arrs['ts_ns'][i] - htf_arrs['ts_ns'][i-1] > 3 * htf_period_ns:
            continue

        # Build HTF candle dicts
        lc  = {k: float(htf_arrs[k][i-1]) for k in ('o','h','l','c')}  # uses open/high/low/close mapped to o/h/l/c
        pc  = {k: float(htf_arrs[k][i-2]) for k in ('o','h','l','c')}
        ppc = {k: float(htf_arrs[k][i-3]) for k in ('o','h','l','c')}

        # Need to map array key names: open→o, high→h, low→l, close→c
        # (handled during array construction or with a helper)

        tspot = detect_tspot(lc, pc, ppc)
        if tspot is None:
            continue

        tspot_type, direction, sweep_mid, close_level, c2_level, c2_src = tspot

        # Touch level = close of T-Spot box
        touch_level = close_level

        # Find chart-TF bars within this HTF period
        htf_start_ns = htf_arrs['ts_ns'][i]         # start of new HTF bar
        htf_end_ns = htf_start_ns + htf_period_ns    # end of this HTF period

        # Actually we need chart bars from when T-Spot formed through next HTF close
        # T-Spot is detected when HTF bar i starts (bar i-1 closed)
        # Chart bars: from htf_start_ns onwards (bars within new HTF candle)
        ct_start = int(np.searchsorted(chart_arrs['ts_ns'], htf_start_ns, side='left'))
        ct_end = int(np.searchsorted(chart_arrs['ts_ns'], htf_end_ns, side='right'))

        if ct_start >= ct_end:
            continue

        # Pivot tracking state
        last_pivot_high = None
        last_pivot_high_bar = -1
        last_pivot_low = None
        last_pivot_low_bar = -1

        # Touch state
        touched = False
        touch_bar_idx = -1

        confirmed = False

        for j in range(ct_start, ct_end):
            # Update pivots (continuous, runs every bar)
            # Pivot at j-2 is confirmed at j (2-bar lookforward)
            if j >= 2:
                ph = detect_pivot_high(chart_arrs['open'], chart_arrs['close'],
                                       chart_arrs['high'], j - 2)
                if ph is not None:
                    last_pivot_high = ph
                    last_pivot_high_bar = j - 2

                pl = detect_pivot_low(chart_arrs['open'], chart_arrs['close'],
                                      chart_arrs['low'], j - 2)
                if pl is not None:
                    last_pivot_low = pl
                    last_pivot_low_bar = j - 2

            # Check touch (if not already touched)
            if not touched:
                if check_touch(float(chart_arrs['high'][j]),
                               float(chart_arrs['low'][j]),
                               float(chart_arrs['open'][j]),
                               float(chart_arrs['close'][j]),
                               touch_level, direction):
                    touched = True
                    touch_bar_idx = j
                continue  # don't check confirmation on same bar as touch

            # Check confirmation (after touch)
            pivot_level = last_pivot_high if direction == 'LONG' else last_pivot_low
            pivot_bar = last_pivot_high_bar if direction == 'LONG' else last_pivot_low_bar

            if pivot_level is None:
                continue

            if check_confirmation(float(chart_arrs['open'][j]),
                                  float(chart_arrs['close'][j]),
                                  pivot_level, pivot_bar,
                                  touch_bar_idx, touch_level, direction):
                confirmed = True

                # Entry = close of confirmation bar
                entry_price = float(chart_arrs['close'][j])
                entry_ts_ns = int(chart_arrs['ts_ns'][j])

                # Stop = extreme between pivot bar and confirmation bar
                if direction == 'LONG':
                    stop_price = float(np.min(chart_arrs['low'][pivot_bar:j+1]))
                else:
                    stop_price = float(np.max(chart_arrs['high'][pivot_bar:j+1]))

                # MAE (risk size from pivot to stop)
                if direction == 'LONG':
                    mae_pct = (pivot_level - stop_price) / pivot_level * 100
                else:
                    mae_pct = (stop_price - pivot_level) / pivot_level * 100

                # MFE — track from entry until stop hit or next HTF close
                mfe_cutoff_ns = htf_end_ns + htf_period_ns  # end of NEXT HTF candle
                mfe_pct = measure_mfe(m1_arrs, entry_ts_ns, entry_price,
                                      stop_price, mfe_cutoff_ns, pivot_level, direction)

                # Build trade row
                trade = {
                    'date': str(htf_arrs['trade_date'][i-1]),
                    'yr': int(htf_arrs['yr'][i-1]),
                    'dow': int(htf_arrs['dow'][i-1]),
                    'hr': int(chart_arrs['hr'][j]),
                    'direction': direction,
                    'tspot_type': tspot_type,
                    'entry_price': round(entry_price, 2),
                    'pivot_level': round(pivot_level, 2),
                    'stop_price': round(stop_price, 2),
                    'sweep_mid': round(sweep_mid, 2),
                    'close_level': round(close_level, 2),
                    'c2_level': round(c2_level, 2) if c2_level else None,
                    'mae_pct': round(mae_pct, 4),
                    'mfe_pct': round(mfe_pct, 4) if mfe_pct is not None else None,
                    'risk_pts': round(abs(entry_price - stop_price), 2),
                }
                trades.append(trade)
                break  # one trade per T-Spot

    return trades
```

```python
def measure_mfe(m1_arrs, entry_ts_ns, entry_price, stop_price,
                cutoff_ns, pivot_level, direction):
    """
    Track MFE from pivot_level on 1m bars after entry.
    Stop tracking when: (a) stop hit, or (b) cutoff_ns reached.
    MFE = max favorable excursion from pivot_level as % of pivot.
    """
    start = int(np.searchsorted(m1_arrs['ts_ns'], entry_ts_ns, side='left'))
    end = int(np.searchsorted(m1_arrs['ts_ns'], cutoff_ns, side='right'))
    end = min(end, len(m1_arrs['ts_ns']))

    if start >= end:
        return None

    if direction == 'LONG':
        max_favorable = pivot_level  # start at pivot
        for k in range(start, end):
            bar_low = m1_arrs['low'][k]
            bar_high = m1_arrs['high'][k]
            # Check stop hit
            if bar_low <= stop_price:
                break
            # Track max high
            if bar_high > max_favorable:
                max_favorable = bar_high
        return (max_favorable - pivot_level) / pivot_level * 100
    else:
        min_favorable = pivot_level  # start at pivot
        for k in range(start, end):
            bar_high = m1_arrs['high'][k]
            bar_low = m1_arrs['low'][k]
            # Check stop hit
            if bar_high >= stop_price:
                break
            # Track min low
            if bar_low < min_favorable:
                min_favorable = bar_low
        return (pivot_level - min_favorable) / pivot_level * 100
```

```python
def agg(g):
    """Aggregate a group of trades — MAE/MFE stats only (no WIN/LOSS)."""
    n = len(g)
    if n == 0:
        return dict(n=0, avg_mae=None, avg_mfe=None, med_mae=None, med_mfe=None)
    mae = g['mae_pct'].dropna()
    mfe = g['mfe_pct'].dropna()
    return dict(
        n=n,
        avg_mae=round(float(mae.mean()), 4) if len(mae) else None,
        avg_mfe=round(float(mfe.mean()), 4) if len(mfe) else None,
        med_mae=round(float(mae.median()), 4) if len(mae) else None,
        med_mfe=round(float(mfe.median()), 4) if len(mfe) else None,
        p90_mae=round(float(mae.quantile(0.90)), 4) if len(mae) >= 5 else None,
        p90_mfe=round(float(mfe.quantile(0.90)), 4) if len(mfe) >= 5 else None,
    )
```

## Important Notes for Implementation

### Array Key Mapping
The HTF arrays from `df_to_arrays()` use keys `open`, `high`, `low`, `close` (not `o`, `h`, `l`, `c`). The `detect_tspot()` function expects `o/h/l/c` keys. Build candle dicts with the right mapping:
```python
lc = {'o': float(htf_arrs['open'][i-1]), 'h': float(htf_arrs['high'][i-1]),
      'l': float(htf_arrs['low'][i-1]),  'c': float(htf_arrs['close'][i-1])}
```

### Resampling for 1M Chart TF
The `15M_1M` model uses 1M as the chart TF. The raw 1M data needs to be wrapped in `df_to_arrays()` format. Use `df_1m_to_arrays()` but ensure it has the same keys (`open`, `high`, `low`, `close`, `ts_ns`, `hr`, `dow`, `trade_date`, `yr`). May need a thin wrapper or alias.

### Pivot Edge Cases
- At the start of a chart-TF window, there may not be enough bars for pivot detection (need j >= ct_start + 2). Handle gracefully.
- Pivots should be tracked continuously across HTF boundaries (not reset per HTF bar) — the `last_pivot_high/low` carries over.

### Gap Detection
Keep the existing gap check: `htf_ts[i] - htf_ts[i-1] > 3 × htf_period_ns` skips weekends/holidays.

### Dashboard Backward Compatibility
The JSON structure changes significantly (no `profiles` wrapper, no `win_rate`/`ev_per_trade`/`profit_factor`). The dashboard will need conditional handling or a full update. Task 8 handles this.
